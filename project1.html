Responsive Website

Resposive app over een duurzaam initiatief

Op telefoon formaat

In dit project heb ik een responsieve website ontworpen voor een duurzaam initiatief gericht op het promoten van groene praktijken en milieubewustzijn. De website richt zich specifiek op de duurzame praktijken van de IJ-hallen, waarbij de nadruk ligt op recycling, hergebruik en milieuvriendelijke evenementen.

Het is een project met de opdracht voor de vakken Vormgeving en Internetstandaarden, heb ik mijn eerste responsieve website gemaakt. Ik gebruikte alleen HTML en CSS om de website te bouwen. Het doel was om een mooie en praktische website te ontwerpen die past bij een duurzaam initiatief. Hiervoor heb ik IJ-Hallen gekozen met de focus op recycling, hergebruik en milieuvriendelijke.

Voor het ontwerp begon ik met schetsen. Dit hielp me om ideeën te bedenken en te kijken wat er goed uitziet. Ik speelde met vormen, kleuren en indelingen totdat ik een ontwerp had dat er niet alleen mooi uitziet, maar ook logisch en makkelijk te gebruiken is. Een belangrijk onderdeel van het ontwerp was dat de website op elk apparaat goed werkt, van computers tot telefoons. Dit heet responsive design. Ik heb veel tijd besteed aan het uitproberen van verschillende indelingen en manieren om de website er overal goed uit te laten zien.

HTML en CSS Bij het bouwen van de website heb ik alleen HTML en CSS gebruikt. Met HTML maakte ik de structuur van de website, zoals koppen, tekst en afbeeldingen. Met CSS gaf ik alles een mooie opmaak, zoals kleuren, lettertypes en de manier waarop de website er op verschillende schermformaten uitziet. Ik heb geleerd hoe ik flexbox en media queries kan gebruiken om ervoor te zorgen dat de website zich automatisch aanpast aan verschillende schermgroottes. Dit betekent dat de indeling van de website verandert afhankelijk van of je op een computer, tablet of telefoon kijkt.

Het thema van de website is duurzaamheid. Ik heb dit terug laten komen in het ontwerp en de inhoud. Bijvoorbeeld: Ik gebruikte natuurlijke kleuren zoals groen en blauw, die passen bij het idee van duurzaamheid. Ik ontwierp iconen die hergebruik en recycling laten zien. Er is een eenvoudige kalender toegevoegd waar bezoekers evenementen over duurzaamheid kunnen bekijken. De teksten op de website leggen uit waarom duurzaamheid belangrijk is en hoe de IJ-hallen bijdragen aan een beter milieu.

Door dit project heb ik geleerd hoe ik een website maak die er goed uitziet én goed werkt. Het was een leuke uitdaging om alles met alleen HTML en CSS te doen. Ik heb vooral veel geleerd over het maken van een ontwerp dat op elk apparaat goed werkt. Ik ben trots op het eindresultaat. De website ziet er niet alleen mooi uit, maar helpt ook om mensen bewust te maken van het belang van duurzaamheid. Dat maakt dit project extra bijzonder!
